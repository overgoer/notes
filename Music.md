# Music — Self-hosted music ecosystem

## Цель

Автоматизированный пайплайн: рекомендации → поиск торрентов → загрузка → чистка тегов → стриминг на iPhone. Раз в неделю — новые релизы в библиотеке.

## Архитектура

```
Локально (macOS)                    Амстердам (aeza)
─────────────────                   ─────────────────
ListenBrainz (статистика)           
       │                            
Exflow (рекомендации → список)      
       │                            
Lidarr (подписки на артистов)       Plex (стриминг)
       │                                 │
Prowlarr (поиск по трекерам)        Plex Amp (iOS)
       │
qBittorrent (загрузка)
       │
MusicBrainz Picard (авточистка тегов)
       │
rsync/syncthing ───────────────────► /music/library
```

### Компоненты

| Компонент | Где живёт | Роль |
|-----------|-----------|------|
| **ListenBrainz** | Локально / облако | Трекинг прослушиваний, открытые рекомендации |
| **Exflow** | Локально | Забирает рекомендации, передаёт в Lidarr |
| **Lidarr** | Локально | Мониторит артистов, ищет новые/нужные релизы |
| **Prowlarr** | Локально | Индексатор — ищет по всем трекерам сразу |
| **qBittorrent** | Локально | Скачивает торренты |
| **MusicBrainz Picard** | Локально | Чистит теги, обложки, переименовывает |
| **rsync/Syncthing** | Локально → Сервер | Синхронизация готовой музыки на сервер |
| **Plex** | Амстердам | Сервер стриминга |
| **Plex Amp** | iPhone | Клиент для прослушивания |

## План реализации

### Фаза 1 — База (локально)

- [ ] Установить Lidarr + Prowlarr + qBittorrent
- [ ] Подключить трекеры к Prowlarr
- [ ] Настроить Lidarr на поиск через Prowlarr
- [ ] Закинуть первых артистов вручную — проверить, что скачивается
- [ ] Настроить MusicBrainz Picard на автопост-обработку

### Фаза 2 — Рекомендации

- [ ] Зарегистрироваться в ListenBrainz
- [ ] Настроить трекинг прослушиваний (через Plex или Navidrome)
- [ ] Установить Exflow
- [ ] Связать Exflow → Lidarr (через API)
- [ ] Проверить еженедельный пайплайн

### Фаза 3 — Синхронизация на сервер

- [ ] Поднять Plex на Амстердаме
- [ ] Настроить rsync/Syncthing между локальным и серверным хранилищем
- [ ] Подключить Plex Amp на iPhone
- [ ] Проверить стриминг

## iOS-клиенты

| Приложение | Цена | Офлайн | CarPlay | Оценка |
|-----------|------|--------|---------|--------|
| **Plex Amp** | Бесплатно (Plex Pass за offline/lyrics) | ✅ | ✅ | ⭐ Лучший выбор. Красивый, умный, из коробки |
| **Ampify** | Бесплатно | ✅ | ✅ | Хороший бесплатный вариант для Navidrome |
| **LMP Music** | $3 | ❌ | ✅ | Красивый, но разработчик принципиально не делает офлайн |
| **Symphonium** | $6 (Android only) | ✅ | ✅ | Не для iPhone, упомянут для контекста |

### Рекомендация

**Plex Amp** — самый простой и красивый путь. Ставишь Plex на сервер, Plex Amp на телефон — и библиотека доступна откуда угодно, с обложками, текстами, плейлистами, CarPlay. За офлайн и тексты нужен Plex Pass (lifetime ~$120, часто бывают скидки до $80).

Если хочешь бесплатно и opensource — Navidrome + Ampify. Без Plex Pass, но интерфейс попроще.
