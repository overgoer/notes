**ТОП-5 API ДЛЯ СТРИМА (с багами)**

**1. Swagger PetStore** — ★★★★☆ (лучший для первого раза)
- POST /pet → можно создать питомца с id=0. API вернёт 200 с id=0. Второй POST с id=0 — перезапишет. Идемпотентности нет.
- POST /pet с телом без обязательных полей → 500 вместо 405/400
- curl: `curl -X POST 'https://petstore.swagger.io/v2/pet' -H 'Content-Type: application/json' -d '{"id": 0, "name": "test"}'` (без category/photoUrls/status — баг: 500, хотя обязательные поля)
- Уровень: низкий. Видно сразу.

**2. JSONPlaceholder** — ★★★★☆
- POST /posts → возвращает 201 Created + id: 101. Но GET /posts/101 → 404. Данные не сохраняются.
- Кейс: разработчик получает 201 и успокаивается, а ресурса не существует.
- curl: `curl -X POST 'https://jsonplaceholder.typicode.com/posts' -H 'Content-Type: application/json' -d '{"title":"foo","body":"bar","userId":1}'` → 201 с id:101. GET /posts/101 → 404.
- Уровень: низкий. Отлично для демонстрации «не доверяй статус-коду».

**3. Restful Booker** — ★★★★☆
- Двойное бронирование: два одинаковых POST /booking с одними датами → 200 OK и разные bookingid. Нет проверки уникальности временного слота.
- curl (дважды): `curl -X POST 'https://restful-booker.herokuapp.com/booking' -H 'Content-Type: application/json' -d '{"firstname":"John","lastname":"Doe","totalprice":150,"depositpaid":true,"bookingdates":{"checkin":"2025-06-01","checkout":"2025-06-05"}}'`
- Два 200, два разных id. Должен быть 409 Conflict.
- Уровень: средний. Хорош для обсуждения бизнес-логики.

**4. GitHub API** — ★★★☆☆
- POST /user/repos без токена → 404 вместо 401 (намеренно, для безопасности). Но баг с точки зрения REST.
- curl: `curl -X POST -H "Accept: application/vnd.github.v3+json" https://api.github.com/user/repos -d '{"name":"test"}'` → 404.
- Уровень: средний. Для обсуждения security-by-obscurity.

**5. OpenWeatherMap** — ★★★☆☆
- Запрос погоды для несуществующего города → 200 с пустыми/дефолтными данными вместо 404.
- curl: `curl 'http://api.openweathermap.org/data/2.5/weather?q=asdfghjkl&appid=YOUR_KEY'` → 200.
- Уровень: средний. Нужен API-ключ (бесплатный).
