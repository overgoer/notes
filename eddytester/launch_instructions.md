# Запуск автоматической настройки VPN с доменом pupupuuu.com

## Текущий статус
✅ **DNS настроен правильно!** 
- Домен `pupupuuu.com` разрешается на IP Cloudflare (`188.114.96.3`, `188.114.97.3` и т.д.)
- Это **нормально** — Cloudflare скрывает реальный IP сервера для маскировки
- Проксирование Cloudflare включено (`proxied: true`)

## Что нужно сделать

### 1. **Скопировать скрипт на сервер**

```bash
# На вашем компьютере (в терминале, из директории с файлами):
scp setup_final.sh root@217.144.185.210:/root/
```

### 2. **Запустить скрипт на сервере**

```bash
# Подключиться к серверу
ssh root@217.144.185.210

# На сервере:
chmod +x /root/setup_final.sh
cd /root
./setup_final.sh
```

### 3. **Следить за выполнением**

Скрипт выполнит:
- Установку и настройку Caddy (веб-сервер)
- Получение SSL сертификата через Cloudflare DNS
- Настройку reverse proxy на Cloak (порт 8443)
- Ротацию SNI каждые 6 часов
- Обновление системы мониторинга

**Время выполнения:** 5-10 минут

## Что произойдёт после запуска

### Через 2-3 минуты:
- Caddy запустится и получит SSL сертификат
- Cloak станет доступен через `https://pupupuuu.com/`

### Проверки после завершения:
```bash
# Проверить TLS сертификат
curl -I https://pupupuuu.com/

# Проверить мониторинг
curl -u admin:/mBZIzZFzdqPdyme http://217.144.185.210:8081/

# Проверить логи Caddy
journalctl -u caddy --no-pager -n 20
```

## Важные примечания

### 1. **Ошибка 521 Cloudflare**
Если сразу после запуска `curl -I https://pupupuuu.com/` возвращает **521** — это нормально. Cloudflare проверяет доступность сервера. Через 1-2 минуты ошибка должна исчезнуть.

### 2. **SSL сертификат**
Caddy автоматически получит SSL сертификат через Cloudflare DNS API. Первый раз может занять до 60 секунд.

### 3. **Безопасность**
После успешной настройки:
```bash
# Удалить скрипт с сервера (содержит API токен)
rm /root/setup_final.sh

# Рекомендуется сменить API токен Cloudflare:
# My Profile → API Tokens → найдите токен → Rotate
```

### 4. **Обновление клиентских конфигураций**
После настройки обновите клиентские конфигурации:
- **Cloak**: сервер `pupupuuu.com`, порт `443`
- **Другие протоколы**: используйте домен вместо IP

## Если возникнут проблемы

### Caddy не запускается:
```bash
systemctl status caddy
journalctl -u caddy -f
```

### Нет SSL сертификата:
```bash
# Проверить DNS разрешение
dig @1.1.1.1 pupupuuu.com

# Проверить API токен
curl -X GET "https://api.cloudflare.com/client/v4/zones" \
  -H "Authorization: Bearer ваш_токен" \
  -H "Content-Type: application/json"
```

### Мониторинг недоступен:
```bash
# Проверить nginx
systemctl status nginx

# Проверить порт 8081
netstat -tlnp | grep 8081
```
