# 🖥 Домашний сервер

**Статус:** 🟢 Планирование

## Концепция
Домашний сервер на имеющихся MacBook — **нулевой бюджет на мозги**.

## Цели
- [x] Файловое хранилище (бэкапы, документы)
- [x] Photo Vault (личное, не Google/iCloud)
- [ ] Резервное копирование устройств (Time Machine + rsync)
- [ ] Docker для экспериментов (потом)
- [ ] Plex/Jellyfin для медиа (потом)

## Оборудование

### Есть (бесплатно)
| Устройство | Роль | Состояние |
|-----------|------|-----------|
| **MacBook Air 2017** | Резерв / тестовый полигон | Работает, 8GB RAM |
| **MacBook Pro 2019** | Основной сервер | 6 ядер, 16GB RAM, мёртвая батарея |

### Нужно докупить (~6000-9000₽)
- [ ] HDD 2-4TB 3.5" (~5000-7000₽)
- [ ] USB-коробка для HDD (~1000-2000₽)
- [ ] USB-C → Ethernet адаптер (~1000₽)

### На вырост (~10000₽)
- [ ] ORICO 9958C3 (5-отсеков, USB 3.2 Gen2) — когда нужно больше дисков

## Настройка

### MacBook Pro 2019 — Первичный сервер
1. Реанимировать (сброс SMC)
2. Отключить батарею (или оставить на зарядке)
3. Чистая установка macOS или Linux (Asahi/Ubuntu)
4. Настройка сервисов:
   - Syncthing (синхронизация файлов)
   - Samba (файлопомойка по сети)
   - Photo Vault (Immich / Photoprism / собственное)
   - Time Machine (бэкапы Mac)

### MacBook Air 2017 — Резерв
1. Чистка и тест
2. Использовать как тестовый стенд

## Ресурсы
- [Asahi Linux](https://asahilinux.org/) — Linux на Apple Silicon / Intel Mac
- [Immich](https://immich.app/) — Photo Vault
- [Syncthing](https://syncthing.net/) — децентрализованная синхронизация
- [ORICO 9958C3](https://aliexpress.ru/item/1005009613655725.html) — 5-отсеков DAS
