# POST без тела — 400 или 500? А если с пустым JSON?

> **POST без тела → 500. Это баг? 🔥**
>
> Реальный кейс из Practicum. Отправляю `POST /api/users` — ловлю 500.
>
> Без тела — 500. С пустым телом `{}` — тоже 500.
>
> **Вопрос: чей это баг?**
>
> 🐛 А) Баг бэкенда — должен отдавать 400 Bad Request
> 🤷 Б) Баг клиента — кто шлёт пустоту, тот идиот
> ✅ В) Не баг — это норм реакция на пустой запрос
>
> 👇 Пиши свой вариант. Через час разберу, кто прав.
>
