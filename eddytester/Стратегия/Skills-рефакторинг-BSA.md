# Skills-рефакторинг BSA

## Дата: 2026-05-20

### Проблема
Монолитный промпт (76 + 41 строка) + 980 строк кода. Всё в контексте сразу → нестабильное форматирование, путаница, шум.

### Решение: модульные скилы
Вместо одного промпта — узкие .md-файлы, загружаемые в начало диалога.

### План (3 шага)
1. **telegram-format** (~20 строк) — всегда. Макс 4000 зн, только дефис, HTML
2. **backlog-rules** (~15 строк) — всегда. Приоритеты, статусы, трекинг
3. **post-writer** (~25 строк) — при создании поста. Готовый текст, не ссылка на Obsidian
