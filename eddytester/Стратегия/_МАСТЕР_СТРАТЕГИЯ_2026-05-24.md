## 3. Открытые вопросы (ждут твоего решения)

### 3.1 Цена Practicum
**Твоё предложение:** 4 990₽
**Контекст:**
- Расходы: сервер Timeweb, поддержка, разработка
- ЦА: Junior QA — готовы платить за практику, но потолок ~3-5к за "тренажёр"
- Конкуренты: курсы 10-60к, но у нас не курс, а симулятор

**Что дальше:** Нужно решение. 4 990 — ок, если продукт = доступ к API + БД + задания. Можно стартануть с 4 990, потом тестить.

### 3.2 Платежная система
**TG Stars:**
- Это да, звёзды Telegram
- Вывод в РФ: через @wallet → на карту, ИП/самозанятость проходит
- Комиссия ~30% — **Эдди прав, много.** С учётом налогов (самозанятость 4-6%) чистыми останется ~64-66% от цены
- Плюс: НЕ нужен сайт, НЕ нужна интеграция с эквайрингом, запуск за день
- Минус: 30% комиссии больно бьют по марже при 4 990₽ (чистыми ~3 300₽)

**Lava.top:**
- Да, на сайте написано от 3.5% до 8% + 10-15₽ за платёж
- Для самозанятых — ок, договор оферты
- Нужен сайт/лендинг для приёма платежей
- Чистыми: ~4 500₽ с продажи 4 990₽ (против 3 300₽ у TG Stars)

**Моя рекомендация:** TG Stars — на старт (быстро, без сайта). Lava.top — через месяц, когда будет лендинг Practicum.

### 3.3 Название продукта
**Сейчас:** API Practicum
**Проблема:** "Practicum" ассоциируется с Яндекс.Практикум. Не описывает суть.

**Варианты для обсуждения:**

| Название | Суть | Минус |
|----------|------|-------|
| **API Practicum** | Текущее | Слияние с Яндекс.Практикум |
| **API-тренажёр** | Понятно, что тренировка | Скучновато |
| **API-полигон** | Звучит экспертно | Менее понятно новичкам |
| **Bugground** | Игра слов Bug + Playground | Англицизм |
| **QA Polygon** | Понятно ЦА | Суховато |
| **API-лаборатория** | Звучит интересно | Длинно |
| **Bug Hunter API** | Продаёт эмоцию | Узко (не только баги) |

**Моя рекомендация:** **API-тренажёр** или **API-полигон** — сразу понятно, что это симулятор для практики. Practicum убираем, чтобы не путать.
