# 📊 API Improvement Tracker

**Предложено → Принято → Сделано → Опубликовано**

## Легенда статусов
| Статус | Значение |
|---|---|
| ⏳ предложено | BSA предложил, ждёт твоего решения |
| ✅ принято | Ты одобрил — можно делать |
| ❌ отклонено | Не вписывается в стратегию сейчас |
| 🔧 в работе | Эдди или Hermes делает |
| 🚀 сделано | Код написан, задеплоено |
| 📢 опубликовано | Вышел пост про это |

## Как пользоваться
1. BSA API Bridge генерирует предложения каждый понедельник
2. Отметь статус в колонке "Статус" — замени `⏳ предложено` на `✅ принято` или `❌ отклонено`
3. Когда сделаешь — `🔧 в работе` → `🚀 сделано`
4. Когда выйдет пост — `📢 опубликовано` + ссылка
