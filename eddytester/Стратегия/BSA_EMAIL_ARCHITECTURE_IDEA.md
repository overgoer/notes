# Идея: Email-канал для сессий с Биззи

## Суть
Постоянный канал связи с Биззи через email. Без терминала, без SSH. Просто пишешь письмо — Биззи отвечает.

## Как работает
1. Тема письма: `bizzy <тема>` (минимум синтаксиса)
2. Биззи проверяет почту каждые 5-10 минут
3. Отвечает письмом с полным разбором, используя все инструменты
4. Пока ты отвечаешь — сессия живёт
5. 10 часов тишины -> снэпшот в Obsidian, сессия archived

## Что есть уже
- gateway.py — IMAP-ридер, читает Gmail, фильтрует eddy.super1
- bsa_agent.py — обрабатывает фидбэк по email
- send_email — отправка писем через Resend
- .mailcfg — Gmail App Password настроен
- Cron IMAP — */30 * * * * (нужно *)5)

## Что нужно сделать
- bsa_email_session.py — полноценный чат через email с полным набором тулов
- Вместо 3 раундов — бесконечная сессия (пока отвечаешь)
- Автоархивация в Obsidian при таймауте

## Формат темы
- Входящее: bizzy <тема>
- Биззи может уточнить тему в ответе
