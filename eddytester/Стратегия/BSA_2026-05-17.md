## Strategic Bets (3-5 на неделю)

### 1. Запуск платёжной интеграции ЮKassa в api-practicum-bot
**Категория:** product | **Effort:** M

Это прямой путь к монетизации API Практикума. Без оплаты продукт остаётся бесплатным, что не поддерживает бизнес. Вердикт PM уже одобрен, реализация критична.

**Ожидаемый эффект:** Появление возможности покупки доступа к v2 (чистой) версии API, начало генерации дохода.

**Критерий успеха:** Пользователи могут оплатить и получить доступ к v2 API через бот.

### 2. Создание лендинга API Practicum
**Категория:** growth | **Effort:** M

Без лендинга воронка продаж не замкнута. Пользователи приходят из Telegram, но не видят структурированного описания продукта и цен. Статический HTML-лендинг на сервере решит эту проблему.

**Ожидаемый эффект:** Повышение конверсии из подписчика в покупателя, чёткое позиционирование продукта.

**Критерий успеха:** Лендинг опубликован, ссылка добавлена в описание канала и бота, отслеживаются переходы.

### 3. Серия постов «Bug Hunt» на основе багов v0-test-api
**Категория:** content | **Effort:** L

v1 API содержит 23 задокументированных бага. Это готовый контент — джуны учатся на реальных примерах. Посты будут вовлекать аудиторию и демонстрировать ценность API.

**Критерий успеха:** Опубликовано 3+ поста, увеличение активности в канале.

### 4. Автоматизация деплоя продуктов через простой скрипт
**Категория:** operations | **Effort:** L

Эдди управляет несколькими репозиториями. Ручной деплой отнимает время и чреват ошибками. Простая автоматизация (bash-скрипт или Node.js) ускорит итерации.

**Ожидаемый эффект:** Сокращение времени на выпуск обновлений, уменьшение операционных ошибок.

**Критерий успеха:** Деплой free-trial-api и v0-test-api выполняется одной командой.
