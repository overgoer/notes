## Practicum (API Practicum) — P0 🏆

**Статус:** В разработке, не запущен
**Монетизация:** Платный курс
**Цена:** Не определена

### Что это
REST API-симулятор для обучения тестированию.
- V1 — с багами (20+ багов)
- V2 — исправленная версия
- 5 эндпоинтов: GET /, GET /:id, POST /, PATCH /, DELETE /

### Стек
- **Сервер:** Express.js, PostgreSQL, PM2, порт 3000
- **Документация:** Swagger (/documentation)
- **Репозиторий:** v0-test-api (на Timeweb)
- **Бот:** Telegram bot (@eddytester bot) — интерфейс для Practicum

### Точки отказа (блокеры)
- [ ] Free Trial API не починен (Postgres auth, эндпоинты)
- [ ] Платёжка не подключена
- [ ] Токен в хардкоде → надо в .env
- [ ] Флоу не пройден лично Эдди

### Критический путь до 10 000₽
1. Free Trial починить
2. Платёжка подключена
3. Весь флоу пройден лично
4. Первый анонс в Telegram
5. Первые продажи

## Telegram Bot (@eddytester bot)

**Статус:** Работает (tg-bizzy, PM2)
**Стек:** Python, DeepSeek v4 Flash API, function calling

### Возможности
- Стратегические консультации (BSA Chat)
- Управление контентом (Content Manager)
- Управление бэклогом
- Оценка задач (PM Agent)
- Запуск ресерчера
- Практикум: баги, подсказки (java-coach)

### Точки отказа
- Работает через long polling (зависимость от сети)
- DeepSeek API может лечь или изменить цены
- Нет мониторинга uptime
