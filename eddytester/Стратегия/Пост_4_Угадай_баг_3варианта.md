# Три ответа API. Один — баг. Найди за 10 секунд.

Ну что, умники, хватит теоркатов. Три реальных ответа API. В одном — баг. Остальные — норма? Ага, щас.

**Кейс А:** Купон 100% на заказ. Сумма корзины — ноль. API возвращает **201 Created**. Платёжка потом списала полную цену. Клиент с товаром и долгом. Всё ок?

**Кейс Б:** Товар давно мёртв — удалён из каталога. POST /orders возвращает **201 Created**. 5000 таких заказов. Логистика встала колом.

**Кейс В:** POST /users возвращает **200 OK**. Заголовка Location нет. Нового юзера не найти. Дубли плодятся сами собой. Классика Spotify, кстати.

---

**Выбери один баг за 10 секунд. Без гугла. Пиши букву в комменты.**

👇 A — ну купон же, бизнес сам разберётся
👇 B — 201 есть 201, товар когда-то существовал
👇 C — 200 это успех, Location не обязателен

Не можешь выбрать? А пользователь уже потерял деньги, товар или ссылку.

---

**Спойлер:** правильного ответа нет. Баги — все три. Кто написал «все три» — тот читал меня раньше. Остальным — думать.
