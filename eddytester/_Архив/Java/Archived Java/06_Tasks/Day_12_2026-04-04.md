### 2. Практика (45 мин) ⏱️

**Задание:** Найди и исправь 5 ошибок в коде ниже.

**Код с ошибками:**
```java
public class DebugExample {
    public static void main(String[] args) {
        int[] nums = {1, 2, 3, 4, 5};
        int sum = 0;
        
        // Ошибка 1: цикл выходит за границы массива
        for (int i = 0; i <= nums.length; i++) {
            sum += nums[i];
        }
        
        // Ошибка 2: деление на 0
        int result = 10 / 0;
        
        // Ошибка 3: неправильный тип переменной
        String name = "Ed";
        int nameLength = name.length(); // ✅
        int nameLength2 = name.length(); // ❌ повторное присваивание без смысла
        
        // Ошибка 4: неинициализированная переменная
        int count;
        System.out.println(count); // ❌
        
        // Ошибка 5: ложное условие
        if (1 == 2) {
            System.out.println("Это всегда выведется"); // ❌
        }
    }
}
```

**Формат ответа:**
- Для каждой ошибки укажи: строку, тип ошибки (runtime, compile-time, logic), исправление.
