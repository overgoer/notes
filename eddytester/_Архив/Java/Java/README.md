**Цель:** Уверенно пройти собеседование AQA Java в Т-Банк / Яндекс / Авито / Ozon / Сбер
**Старт:** Май 2026
**Режим:** 2 часа/день, воскресенье — выходной
**Фокус:** Java Core, многопоточность, инструменты тестирования (без деревьев/DP/графов)

| Фаза | Недели | Тема | Контрольная точка |

## Ежедневная рутина (2 часа)

| Время | Что делаем |
|-------|-----------|
| 0:00-0:05 | Разминка — открыть конспект, вспомнить |
| 0:05-0:35 | Практика (CodingBat / LeetCode) — 2-3 задачи |
| 0:35-1:20 | Теория — 1 тема, конспект в Obsidian |
| 1:20-1:30 | Перерыв |
| 1:30-1:55 | Закрепление — переписать код по памяти |
| 1:55-2:00 | Итоги — оценка (1-5), план на завтра |

> **Важно:** Фаза 0 — не прыгать дальше, пока массивы, строки и циклы не станут автоматом.

---

## Принципы

1. **Минимум 2 часа, максимум 3** — лучше остановиться на подъёме
2. **Воскресенье — выходной** — мозгу нужно усваивать
3. **Каждый день — маленький результат** — решённая задача + конспект
4. **CodingBat — каждый день** в Фазе 0 и 1
5. **Раз в 2 недели — ревью прогресса**
