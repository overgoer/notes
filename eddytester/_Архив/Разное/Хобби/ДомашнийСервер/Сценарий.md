# 📋 Сценарий использования

## Философия
- **0₽ на мозги** — MacBook Pro 2019 как сервер
- Только необходимое оборудование на старте
- Расширяться по мере необходимости
- Беречь Taycan 🚗

## Почему не Mini PC / NAS
- MacBook Pro 2019 (6 ядер, 16GB) мощнее бюджетного Synology
- Два MacBook — встроенный резерв
- Батарея = UPS (если оставить)
- Можно начать сегодня, не ждать доставки

## Архитектура

```
Интернет
   │
   ├── MacBook Pro 2019 ─── USB-C Ethernet ─── Роутер
   │        │
   │        ├── USB-коробка ─── HDD 2-4TB (данные, бэкапы)
   │        ├── Photo Vault (иммич/фотопризм)
   │        ├── Syncthing (синхронизация)
   │        └── Docker (потом)
   │
   └── MacBook Air 2017 ─── резервный сервер
```

## Фото Vault
Личное хранилище фотографий. Вместо Google Photos / iCloud.

**Варианты:**
- **Immich** — современный, похож на Google Photos (рекомендую)
- **Photoprism** — зрелый, мощный
- **Piwigo** — простое, лёгкое

Выбор сделаем при настройке.

## Бэкапы
- Time Machine → внешний HDD
- rsync → второй диск
- Критичные данные → ещё куда-то (S3/другое устройство)

## Развитие
Когда MACBOOK перестанет хватать (мало памяти/дисков/CPU) → Mini PC b/у + ORICO.
