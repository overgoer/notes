## 4. БАНК → ДРУГИЕ ДОМЕНЫ

✅ Согласовано. Все отсылки к банку убираем.

**Новые домены для сериалов и кейсов:**
1. **E-commerce** — интернет-магазин (корзина, заказы, оплата, скидки). Самый понятный контекст для джунов.
2. **SaaS** — CRM или платформа для подписок (логика биллинга, тарифов, ролей).
3. **Логистика** — доставка, трекинг, складские остатки.
4. **Геймдев** — игровой API (лидерборды, матчмейкинг, инвентарь).

**Что меняем в 44 постах:**
- Сериал «Мне дали API банка» → «Мне дали API интернет-магазина» (10 серий)
- Все упоминания «банк» → «e-commerce/SaaS/логистика»
- Баги (отрицательный перевод, баланс) → баги e-commerce (отрицательная цена, двойной заказ)
- Security-баги — остаются (CORS, rate limiting, авторизация — универсально)

**Как переработать:** берём сериал (4 готовые серии) и переписываем контекст. Не надо менять суть багов — меняем декорации. Отрицательный перевод → отрицательная цена товара. Баланс = остаток на складе.
