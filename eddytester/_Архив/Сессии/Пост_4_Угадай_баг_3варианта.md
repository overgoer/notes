# Эта компания вернула 200 OK там, где должна была вернуть 401. Угадай, кто?

Я отправляю GET-запрос с невалидным токеном.

Ожидаю: `401 Unauthorized` + сообщение об ошибке.

Получаю: `200 OK` + пустое тело `{}`.

Клиент думает «запрос успешен». Данные не пришли, но никто не заметил. Сторонние интеграции работают в анонимном режиме, пользователи видят пустые профили, кеш отравлен.

Этот баг висел ГОДАМИ. Не день, не неделя — годы. Тысячи разработчиков просто не заметили, потому что проверяли только статус-код.

👇 **C** — Bitbucket
