## Strategic Bets (3-5 на неделю)

### 1. Запуск платёжной интеграции ЮKassa в api-practicum-bot
**Категория:** product | **Effort:** M (1-2 дня на интеграцию, тестирование и деплой)

Это самый прямой путь к монетизации — без платежей воронка продаж не работает. PM Agent уже одобрил задачу, нужно выполнить. Интеграция даст возможность продавать доступ к v2 API Practicum и сразу начнёт приносить данные о конверсии.

**Ожидаемый эффект:** Появится работающий платёжный поток, можно тестировать ценообразование и собирать первых платящих пользователей.

**Критерий успеха:** Пользователь может оплатить подписку через бота и получить доступ к v2 API.

### 2. Создание лендинга для API Practicum с описанием воронки
**Категория:** growth | **Effort:** M (1-2 дня на базовую страницу с HTML/CSS, интеграция с ботом)

Без лендинга продукт не имеет точки входа для новых пользователей. Лендинг должен объяснять ценность (REST API с багами/без), показывать примеры и вести к пробному периоду или покупке. Это повысит конверсию из канала в продукт.

**Ожидаемый эффект:** Увеличится трафик на продукт, появится возможность собирать email/контакты, улучшится SEO.

**Критерий успеха:** Лендинг опубликован, ссылка добавлена в описание канала, первые переходы на продукт из лендинга.

### 3. Выпуск серии постов по POST/PUT/DELETE на основе free-trial-api
**Категория:** content | **Effort:** M (1 день на подготовку 3-4 постов)

Аудитория ждёт контента по этим методам — это прямой запрос из описания. free-trial-api уже содержит эндпоинты, можно снять скриншоты/видео тестов и написать гайды. Контент закроет ключевой пробел и привлечёт новых подписчиков.

**Ожидаемый эффект:** Повышение вовлечения и удовлетворённости текущей аудитории, приток новых джунов.

**Критерий успеха:** 3 поста опубликованы в @eddytester, реакции и комментарии выше среднего.

### 4. Внедрение tgstat для аналитики канала
**Категория:** operations | **Effort:** L (2-3 часа на установку Docker-контейнера и настройку)

Одна из проанализированных wishlist-идей — самая полезная для операционки. tgstat даст метрики по просмотрам, охвату, динамике подписок. Это позволит принимать решения на основе данных, а не интуиции. Развёртывание через Docker займёт несколько часов.

**Ожидаемый эффект:** Появится дашборд с ключевыми KPI канала, возможность отслеживать эффективность контента и выявлять лучшие форматы.

**Критерий успеха:** Дашборд доступен по ссылке, отображает данные за последние 7 дней.

### 5. Настройка еженедельного дайджеста контента (teledigest)
**Категория:** content | **Effort:** M (1 день на интеграцию с контент-пайплайном и настройку расписания)

teledigest — ещё одна wishlist-идея, которая автоматизирует создание еженедельного дайджеста для канала. Это сэкономит время на ручном сборе материалов и повысит ценность для подписчиков. Легко реализовать на базе blog-analysis.

**Ожидаемый эффект:** Подписчики получают структурированную подборку лучших постов за неделю, что увеличивает удержание и репосты.

**Критерий успеха:** Дайджест автоматически публикуется каждую неделю, открываемость >20%.
