## 5. Эволюция стиля

### Ранние посты (POST 4 — POST 50, ~2023)
- **Преимущественно полезный контент:** статьи, гайды, инструменты
- **Минимум личного:** посты в основном информационные
- **Минимум мемов:** только изредка #memes
- **Короткие подписи:** «Читать на Zen», «(No comments)»
- **Без призывов к действию:** редко задаёт вопросы аудитории
- **Стиль:** более формальный, академичный

### Средние посты (POST 50 — POST 200, 2023-2024)
- **Запуск подкаста iТёрка** — переломный момент. Появляются анонсы выпусков
- **Увеличение доли личного:** истории про карьеру, интервью, выступления
- **Рост мемов/юмора:** регулярные мем-посты, «Жиза»
- **Вовлечение аудитории:** начинает задавать вопросы в конце постов
- **Формирование бренда:** постоянные гости, своё место (iТёрка)
- **Первая монетизация:** упоминание Boosty

### Поздние посты (POST 200 — POST 383, 2024-2025)
- **Зрелый стиль:** уверенный, с юмором, менее формальный
- **Больше «про себя»:** отпуск, книги, скалодром, день рождения, командировка
- **Практикум по API:** длинная серия постов (345→383) с анонсами, прогрессом
- **Разделение каналов:** @eddytester для QA, @iterka для подкаста
- **Эфиры/стримы:** регулярные прямые эфиры (с Цакуновым, ревью резюме)
- **«Библиотека тестировщика»:** читательский клуб (новая рубрика)
- **Boosty:** активная монетизация (дайджесты, переводы статей, полные версии)
- **Глубокие технические посты:** GET с телом, длина URL, 400-ки

**Ключевой переход:** от «полезного блога для QA» → «личного бренда с подкастом, мерчем и сообществом». Стиль стал более человечным, разговорным.

## 7. Что не работало — посты без комментов, слабые темы

**Всего без комментариев: 96 постов (44%)**
**С комментариями: 123 поста (56%)**

### Паттерны «холодных» постов:

1. **Гайды без вопросов в конце** — если пост просто «вот инструмент/статья/список», комментариев почти нет (POST 18, 22, 65, 72, 118, 184, 208, 229, 272, 273, 303)

2. **Личные посты про путешествия/хобби** — командировка в Минск (POST 354), отпуск (POST 364), книги (POST 364), Майские в Перми (POST 143) — почти без отклика

3. **Чистые новости/реакции** — Apple сберегательные счета (POST 43), Opera с ChatGPT (POST 38), Baidu такси (POST 171) — 0-1 комментарий

4. **Короткие мемы без вопроса** — некоторые мем-посты получают комменты, но многие проходят «в тишине» (POST 44, 80, 81, 206, 360)

5. **Просто ссылки на статьи/видео** — «Читать на Zen», «Смотреть тут» без вовлечения аудитории (POST 4, 7, 92, 246)

6. **Анонсы в конце серии** — когда тема уже "разжёвана", последние анонсы получают меньше внимания (POST 383 vs POST 345)

### Что НЕ стоит публиковать (слабые темы):
- Чисто новостные перепосты без авторского комментария
- «Голые» ссылки на сторонние статьи
- Короткие мемы без дискуссионного вопроса
- Персональные лайфстайл-посты (если не задан вопрос)

## Резюме и инсайты

### Что нового узнал:

1. **Отношение комментариев к постам без комментов ≈ 1.28:1** — больше половины постов получают отклик, что для Telegram-канала про QA отличный показатель
2. **Практикум по API — самый вовлекающий проект:** 45+ комментариев на пост-анонс, десятки «+», живой интерес
3. **Подкаст iТёрка — главный драйвер роста:** запуск подкаста изменил стиль канала с «полезный блог» на «личный бренд»
4. **Аудитория — начинающие/мидл QA:** запрос на карьерные советы, поиск работы, инструменты
5. **Эд сознательно диверсифицирует контент:** полезные посты + мемы + личное = живой канал

### Инсайты:

- **🔑 Вовлечение требует призыва к действию:** посты с вопросом в конце получают в 3-5 раз больше комментариев
- **🔑 Личные истории > сухие гайды:** пост про первый баг (POST 363) получил 15 комментов — больше, чем многие инструментальные гайды
- **🔑 Серийность работает:** цикл «анонс → опрос → прогресс → запуск» для практикума дал стабильное вовлечение на протяжении 8 постов
- **🔑 Разделение контента (QA vs подкаст) — логичный шаг:** два канала позволяют не терять подписчиков, которым интересна только одна тема
- **🔑 Страх аудитории перед ИИ — это возможность:** посты, адресующие страх (POST 329-331), получают сильный отклик
- **🔑 Boosty как вторая нога:** монетизация началась органично — сначала дайджесты, потом переводы статей, потом ревью резюме
