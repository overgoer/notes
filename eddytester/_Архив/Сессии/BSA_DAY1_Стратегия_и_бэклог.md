## 2. СТРАТЕГИЧЕСКИЕ СТАВКИ (bets)

### Bet #1: Practicum — единственный продукт (P0)
**Что делаем:** Доделываем P0, запускаем продажи. Ничего другого не начинаем.
**Ожидание:** Первые 10 продаж за 2 недели после запуска.
**Метрика:** Конверсия бот → покупка. Цель: 3-5%. Выручка: 50 000₽ в первый месяц.
**Риск:** Платёжка не готова, технические баги оттолкнут первых покупателей.
**Статус:** 🟢 ЕДИНСТВЕННАЯ СТАВКА. Всё остальное — инструменты для неё.

### Bet #2: Контент = трибуна для Practicum (инструмент)
**Что делаем:** Каждый пост отвечает на вопрос "как это поможет продать Practicum?". 44 готовых поста — бери и публикуй.
**Ожидание:** Рост переходов из TG в бот. Цель: 20% подписчиков канала в месяц заходят в бот.
**Метрика:** Переходы по ссылке на бот / охват поста.
**Риск:** "Продающие" посты собьют лояльность. Митигация: 40/40/20 — вовлечение/обучение/продажа.

### Bet #3: Вовлечение через механику (инструмент)
**Что делаем:** Каждый третий пост — с механикой (угадай баг, исправь меня, да/нет). Без механики — только имиджевые разборы.
**Ожидание:** Доля постов с комм. > 60% (сейчас 56%).
**Метрика:** % постов с >5 комментариями.
**Риск:** Механики надоедят. Митигация: чередовать 3-4 разных механики из 10 форматов.

### Bet #4: Контент-система работает (инструмент)
**Что делаем:** Использовать готовые 44 поста + 3 темы из пула ресерчера в неделю. Не коллекционировать темы.
**Ожидание:** +3 поста в неделю без ручного ресерча.
**Метрика:** Использовано тем / добавлено новых.
**Риск:** AI-текст без личности Эда. Митигация: GA выдаёт факты, Эд пишет голосом.
