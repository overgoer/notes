# <span style="color:#ff4d4d">ТЕХНИЧЕСКОЕ «МЯСО»</span>

* **Триггер:** `Ctrl + Click` по Markdown-ссылке. 
* **Функция:** <code style="color:#ff4d4d">ShellExecuteExW</code>. 
* **Суть:** Notepad скармливает этой функции **всё**, что зашито в ссылке. `ShellExecuteExW` превращает обычную строку текста в **исполняемую команду**. Благодаря этой «фиче» клик по ссылке в Блокноте теперь технически мало чем отличается от запуска сомнительного `.exe` файла.

# <span style="color:#ff4d4d">ATTACK: ПЕРЕХВАТ ВЛАСТИ</span>

* **Действие:** Злоумышленник удаляет владельца через `POST /tuya/removeDevice`.

# <span style="color:#4da6ff">LG SMART TV: ДЫРЯВЫЙ ЭКРАН</span>

* **Фича:** Доступ к файлам ТВ через API (фото/документы). 
* **Ошибка:** Параметр `getFile` никак не валидируется. 
* **Техника:** Старый добрый **Directory Traversal** (`../../`).
