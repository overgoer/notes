# Инструкция по настройке домена и Cloudflare для VPN

## 1. Регистрация домена pupupuuu.com на Porkbun

1. Перейдите на сайт [porkbun.com](https://porkbun.com)
2. Введите `pupupuuu.com` в поле поиска домена
3. Добавьте домен в корзину и завершите покупку ($11/год)
4. При регистрации укажите:
   - Реальные данные (для WHOIS можно включить privacy)
   - Email для подтверждения
   - Способ оплаты (карта, PayPal)
5. После оплаты домен появится в вашем аккаунте Porkbun

## 2. Создание аккаунта Cloudflare (если нет)

1. Перейдите на [cloudflare.com](https://cloudflare.com)
2. Нажмите "Sign Up" и создайте аккаунт (используйте email)
3. Подтвердите email (проверьте почту)

## 3. Добавление сайта в Cloudflare

1. В панели Cloudflare нажмите "Add a Site"
2. Введите `pupupuuu.com` и нажмите "Add Site"
3. Выберите бесплатный план (Free) → "Continue"
4. Cloudflare просканирует существующие DNS записи (их пока нет)
5. На следующем экране вы увидите **2 NS сервера**, например:
   ```
   lara.ns.cloudflare.com
   marty.ns.cloudflare.com
   ```
   **Запишите их!**

## 4. Изменение NS серверов в Porkbun

1. Войдите в аккаунт Porkbun
2. Перейдите к управлению доменом `pupupuuu.com`
3. Найдите раздел "Nameservers" или "DNS Management"
4. Измените NS серверы на предоставленные Cloudflare:
   - Удалите существующие NS записи
   - Добавьте два новых NS сервера от Cloudflare
5. Сохраните изменения

## 5. Ожидание распространения DNS

- Изменение NS серверов может занять **от 1 до 48 часов** (обычно 1-2 часа)
- Проверить статус можно командой (в терминале):
  ```bash
  dig NS pupupuuu.com
  ```
  Должны отображаться Cloudflare NS серверы
- В панели Cloudflare статус сайта изменится с "Pending" на "Active"

## 6. Настройка DNS записей в Cloudflare

После активации сайта в Cloudflare:

1. Перейдите в раздел "DNS" → "Records"
2. Добавьте A-запись для прямого доступа:
   - Type: `A`
   - Name: `@` (или оставьте пустым для корневого домена)
   - IPv4 address: `217.144.185.210` (IP вашего сервера)
   - Proxy status: **Оранжевое облако** (проксирование включено)
   - TTL: Auto

3. (Опционально) Добавьте запись для поддомена:
   - Type: `A`
   - Name: `vpn` (для vpn.pupupuuu.com)
   - IPv4 address: `217.144.185.210`
   - Proxy status: **Оранжевое облако**
   - TTL: Auto

## 7. Создание API токена Cloudflare

Для автоматической настройки сервера потребуется API токен:

1. В панели Cloudflare: My Profile → "API Tokens"
2. Нажмите "Create Token"
3. Выберите шаблон "Edit zone DNS"
4. В разделе "Zone Resources" выберите зону `pupupuuu.com`
5. Продолжите и создайте токен
6. **Скопируйте и сохраните токен в безопасном месте!**

## 8. Проверка доступности домена

После распространения DNS:

1. Проверьте, что домен разрешается на ваш IP:
   ```bash
   dig pupupuuu.com
   ```
   Должен вернуть `217.144.185.210`

2. Проверьте проксирование Cloudflare:
   ```bash
   curl -I https://pupupuuu.com
   ```
   Должна быть ошибка SSL (так как сертификат ещё не настроен), но заголовки должны показывать Cloudflare

## Примечания

- Не отключайте проксирование (оранжевое облако) для A-записей VPN сервисов
- Cloudflare бесплатный план включает CDN, DDoS защиту и автоматический SSL
- Для работы WebSockets (Shadowsocks с v2ray-plugin) убедитесь, что в Cloudflare включена опция "WebSockets" (по умолчанию включена)
- Если возникнут проблемы с доступом через домен, можно временно использовать прямой IP
