# JSON syntax

**Просмотры:** 10 000

**Формат:** Обучающее короткое видео

## Субтитры

Какие у JSON есть правила? У JSON есть **6 типов данных**:

1. **String** — любой текст в кавычках
2. **Number** — число (с точкой, без точки, отрицательное)
3. **Boolean** — всего два варианта: true или false
4. **Null** — пустота
5. **Array** [ ] — для перечисления. Элементом массива может быть любой другой тип данных
6. **Object** { } — когда хотим подробно описать что-нибудь. Внутри поля: слева имя, справа значение, между ними двоеточие
