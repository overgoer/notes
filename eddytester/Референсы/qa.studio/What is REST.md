# What is REST?

**Просмотры:** 5 800

**Формат:** Объяснение основ

## Субтитры

Что такое REST? Это архитектурный стиль, рекомендация, договорённость, философия, соглашение. То есть это свод правил, как реализовать HTTP общение между клиентом и бэкендом.

REST — это свод рекомендаций. "Как было бы здорово всем в этом мире, если мы будем все придерживаться таких-то правил". Кто хочет — придерживается, кто хочет — нет. Но на практике все стараются придерживаться.

Основные типы HTTP запросов (которые используются everyday): **GET, POST, PUT, DELETE, PATCH**.

Редкие: OPTIONS, TRACE, HEAD, CONNECT. Это всё HTTP запросы, но у каждого своя особенность.
