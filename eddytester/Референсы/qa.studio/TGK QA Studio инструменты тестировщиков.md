## Субтитры

Какими инструментами пользуются тестировщики в этом году? Смотрим на данные, которые собираем сами.

**TMS (тест-менеджмент системы):**
1. 🥇 **Test IT**
2. 🥈 **Allure TestOps**
3. 🥉 **Зефир** (внутри Жиры)
4. **TestRail** — по-прежнему
5. **Case.io** — по-прежнему

**Архитектурный стиль/протокол:**
- **REST** занимает огромным отрывом 1-е место
- Все остальные — 5-8%

**Кто выкатывает сборку?**

- 20% — есть CD и тестировщик сам управляет выкадкой
- Большинство — только разработчик

**Мессенджеры:**
