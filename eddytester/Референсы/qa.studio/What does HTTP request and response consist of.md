# What does an HTTP request and response consist of?

**Просмотры:** 8 100

**Формат:** Объяснение основ

## Субтитры

HTTP запрос состоит из трёх частей:
1. **Кому** (адрес)
2. **Технические свойства, метаданные** (headers)
3. **Само сообщение** (body)

HTTP ответ тоже состоит из трёх частей:
1. **Body** — для обмена информацией
2. **Headers** — те же заголовки (есть и в запросе, и в ответе)
3. **Статус** — новое поле, которого не было в запросе
