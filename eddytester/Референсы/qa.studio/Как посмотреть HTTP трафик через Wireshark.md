# Как посмотреть HTTP трафик через Wireshark

**Просмотры:** 6 300

**Формат:** Совет для собеседования

## Субтитры

На собеседовании, когда рассказывал, чем отличается HTTPS от HTTP, задали вопрос: "Понятно, что HTTPS более безопасный, а как это протестировать?"

Есть такой инструмент **Wireshark**. Он показывает передачу всего трафика: рукопожатия, пакеты, TCP, UDP соединения.

Можно открыть сайт через HTTP и через HTTPS параллельно и смотреть, как передаются данные. В HTTP можно разглядеть содержимое запроса. Если открываете сайт через HTTPS — данные будут зашифрованы, и вы не сможете их прочитать.

От вас не требовали прям шарить экран и показывать — достаточно сказать слово **Wireshark**, что через Wireshark можно посмотреть, что HTTPS шифрует, а HTTP не шифрует.
