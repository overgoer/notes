# Free Trial API — Документация

> Бесплатный API для знакомства с тестированием.
> Попробуй, найди баги — и реши, нужен ли тебе полный курс.

## Базовые требования

- **URL:** `http://85.193.81.51:3000/free/api`
- **Формат запросов:** JSON
- **Заголовок авторизации:** `X-Fix-Bug: free-trial`
- **Content-Type заголовок:** `application/json`

## Лимиты

| Тип лимита | Значение |
|-----------|:--------:|
| Запросов в минуту | **10** |
| Максимум пользователей | 100 на один API-ключ |
| Время жизни данных | 24 часа (автоочистка) |
| Одновременных соединений | 5 с одного IP |

**Превышение лимита:** сервер вернёт `429 Too Many Requests`.
Подожди 1 минуту и попробуй снова.

## POST /free/api/users — Создать пользователя

Создаёт тестового пользователя в бесплатном окружении.

### Запрос

```http
POST /free/api/users
Content-Type: application/json
X-Fix-Bug: free-trial

{
  "name": "John Doe",
  "age": 25
}
```

### Параметры

| Поле | Тип | Обязательное | Описание | Пример |
|------|-----|:-----------:|----------|--------|
| name | string | ✅ | Имя пользователя | `"John Doe"` |
| age  | integer | ✅ | Возраст | `25` |

### Успешный ответ

```http
HTTP/1.1 201 Created
Content-Type: text/plain

{
  "id": 1,
  "name": "  John Doe  ",
  "age": 25,
  "status": "candidate"
}
```

### Ошибки

| Код | Описание |
|:---:|----------|
| 400 | name или age не переданы |
| 429 | Превышен лимит запросов (10/мин) |

## GET /free/api/users — Получить список пользователей

Возвращает список пользователей, созданных через Free Trial API.

### Запрос

```http
GET /free/api/users
X-Fix-Bug: free-trial
```

### Параметры (query)

| Параметр | Тип | Обязательное | Описание | Пример |
|----------|-----|:-----------:|----------|--------|
| sort | string | нет | Поле для сортировки | `sort=name`, `sort=-id` |
| limit | integer | нет | Количество записей | `limit=5` |
| status | string | нет | Фильтр по статусу | `status=candidate` |

### Успешный ответ

```http
HTTP/1.1 200 OK
Content-Type: text/plain

[
  {
    "id": 1,
    "name": "  John Doe  ",
    "age": 25,
    "status": "candidate"
  }
]
```

### Ошибки

| Код | Описание |
|:---:|----------|
| 429 | Превышен лимит запросов (10/мин) |

## Примеры запросов (curl)

```bash
# 1. Создать пользователя
curl -X POST http://85.193.81.51:3000/free/api/users \
  -H "Content-Type: application/json" \
  -H "X-Fix-Bug: free-trial" \
  -d '{"name":"Alice","age":25}'

# 2. Создать с лишними пробелами в имени
curl -X POST http://85.193.81.51:3000/free/api/users \
  -H "Content-Type: application/json" \
  -H "X-Fix-Bug: free-trial" \
  -d '{"name":"  Bob  ","age":30}'

# 3. Создать с отрицательным возрастом
curl -X POST http://85.193.81.51:3000/free/api/users \
  -H "Content-Type: application/json" \
  -H "X-Fix-Bug: free-trial" \
  -d '{"name":"Charlie","age":-5}'

# 4. Получить всех
curl http://85.193.81.51:3000/free/api/users \
  -H "X-Fix-Bug: free-trial"

# 5. Попробовать сортировку и лимит
curl "http://85.193.81.51:3000/free/api/users?sort=id&limit=10" \
  -H "X-Fix-Bug: free-trial"

# 6. Фильтр (регистр)
curl "http://85.193.81.51:3000/free/api/users?status=MINOR" \
  -H "X-Fix-Bug: free-trial"
```

## Очистка данных

Бесплатное окружение автоматически очищается раз в 24 часа.
Если нужно сбросить данные раньше — просто подожди или создай нового пользователя (старые всё равно удалятся при очистке).
