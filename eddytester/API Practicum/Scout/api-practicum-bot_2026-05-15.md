## Findings (4)

### 🔴 Интеграция YooKassa для платного доступа к лабораторным

**Priority:** high | **Effort:** days

Добавить приём платежей через YooKassa. Это позволит продавать доступ к лабораторным работам (balance_lab, task5 и будущим), создав прямой канал монетизации без ручных консультаций. Соответствует стратегической ставке BSA.

### 🔴 Контентная серия по POST/PUT/DELETE и Auth на основе свободного API

**Priority:** high | **Effort:** days

Создать в боте серию заданий по методам HTTP и авторизации, используя существующий free-trial-api. Это привлечёт джунов, увеличит вовлечение и контент для @eddytester. Соответствует BSA.

### 🟡 Разобраться с причиной revert фичи task5 host header lab

**Priority:** medium | **Effort:** hours

Коммит 'Revert feat: add task5 host header lab' указывает на возможные баги или конфликты. Необходимо проанализировать код, исправить и повторно добавить фичу. Это важно для целостности контентного пайплайна.

### 🟡 Автоматизация деплоя (CI/CD) для быстрых релизов

**Priority:** medium | **Effort:** hours

Настроить простой pipeline (например, GitHub Actions) для автоматического деплоя на сервер. Сэкономит время Эда на рутинных операциях и уменьшит риск ошибок.


## Relevance to BSA
Findings 1 и 2 напрямую реализуют стратегические ставки: интеграция YooKassa запускает платёжный флоу, а контентная серия даёт материал для @eddytester и привлекает джунов. Finding 3 (revert) может блокировать контентную серию, если task5 использовалась как основа. Finding 4 повышает operational efficiency, что косвенно ускоряет поставку контента и продукта.
