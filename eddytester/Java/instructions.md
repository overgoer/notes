## 🎯 **Программа подготовки: 8-недельный план до уровня Senior QA (Java)**

### **Неделя 1-2: Java Core Foundation**
**Цель:** Уверенное владение основами Java, способность писать чистый код без IDE.
**Ежедневный ритуал:**
- 30 минут: CodingBat (задачи на массивы, строки, логику)
- 60 минут: Изучение Core Java по Хорстманну (главы 1-5)
- 30 минут: Практика на доске (3 микро-задачи от интервьюера)
- 30 минут: Конспектирование в Obsidian (папки 01_Interview_Questions, 03_Theory_Core)

**Контрольные точки:**
- День 3: Разбор == vs equals(), hashCode(), иммутабельность строк
- День 5: Коллекции (List, Set, Map) — время доступа, когда что использовать
- День 7: Исключения (checked/unchecked), try-with-resources
- День 10: Дженерики, wildcards
- День 14: Модель памяти Java (стек/куча), final, static

### **Неделя 3-4: Алгоритмы и структуры данных для QA**
**Цель:** Умение оценивать сложность алгоритмов, решать задачи LeetCode Easy/Medium.
**Ежедневный ритуал:**
- 45 минут: Neetcode.io (раздел Arrays & Hashing)
- 45 минут: Решение 2 задач LeetCode (без IDE)
- 30 минут: Разбор Big O для каждого решения
- 30 минут: Конспектирование в 02_LeetCode_Drills

**Контрольные точки:**
- День 16: Поиск, сортировка (бинарный поиск, QuickSort)
- День 18: Связные списки, двусторонние очереди
- День 21: Деревья (обход в глубину/ширину)
- День 25: Графы (базовые алгоритмы)
- День 28: Динамическое программирование (простые задачи)

### **Неделя 5-6: Продвинутые темы Java и QA-специфика**
**Цель:** Глубокое понимание многопоточности, IO, тестирования.
**Ежедневный ритуал:**
- 60 минут: Effective Java (главы 2, 3, 4)
- 45 минут: Практика многопоточных сценариев
- 45 минут: Написание тестов (JUnit 5, Mockito)
- 30 минут: Разбор реальных кейсов из BigTech

**Контрольные точки:**
- День 30: Многопоточность (synchronized, volatile, ConcurrentHashMap)
- День 33: Паттерны проектирования (Singleton, Factory, Builder)
- День 36: Работа с файлами, сетью (Java NIO)
- День 39: Аннотации, рефлексия
- День 42: Spring Boot основы (для понимания контекста)

### **Неделя 7-8: Интервью-симуляции и полировка**
**Цель:** Полная готовность к реальным интервью.
**Ежедневный ритуал:**
- 60 минут: Mock-интервью (кодирование на доске)
- 60 минут: Разбор поведенческих вопросов (STAR метод)
- 60 минут: Анализ сливов интервью (2024-2025)
- 30 минут: Работа над слабыми местами (по оценкам из 04_Mock_Sessions)

**Контрольные точки:**
- День 44: Полное mock-интервью (2 часа)
- День 47: Финальный разбор всех тем
- День 50: Составление чек-листа для интервью
- День 56: Финальная оценка — готовность к офферу

## 🔄 **Гибкость программы**
Если в какой-то день Ed не усваивает тему:
1. **Снизить нагрузку:** Убрать 1 блок практики, добавить 30 минут повторения.
2. **Изменить подход:** Использовать альтернативные ресурсы (metanit.com, видео).
3. **Взять паузу:** Повторить предыдущие темы, закрепить фундамент.

Программа пересматривается каждую неделю на основе оценок из **04_Mock_Sessions**.
