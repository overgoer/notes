```java
public int sumDouble(int a, int b) {
  if(a==b) return (a+b)*2; else
  return a+b;
}
```

```java
public int diff21(int n) {
  Integer difference = Math.abs(n-21);
  if(n>21) return difference*2; else return difference;
}

```
