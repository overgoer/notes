### 2. Практика (45 мин) ⏱️

**Создай класс `Calculator` с нуля (без IDE).**

**Требования:**
1. Метод `int add(int a, int b)` — возвращает сумму.
2. Метод `int multiply(int a, int b)` — возвращает произведение.
3. Метод `double divide(double a, double b)` — возвращает частное (обрати внимание на деление на 0).
4. Метод `int max(int a, int b)` — возвращает максимальное из двух.
5. **Перегрузи** `max` для трёх аргументов: `int max(int a, int b, int c)`.

**Напиши код в этом файле (в разделе для ответов).**

**Проверка:**
- Вызови все методы на примере: `a = 10, b = 3, c = 7`.
- Убедись, что `divide(10, 0)` возвращает `Double.NaN` или выбрасывает `IllegalArgumentException`.

### 3. Теория (30 мин) ⏱️

**Источник:** [Metanit: Параметры и возврат значений](https://metanit.com/java/tutorial/2.10.php) и [Перегрузка методов](https://metanit.com/java/tutorial/2.11.php)

**Прочитай и законспектируй:**
- Параметры: передача по значению (для примитивов и ссылок).
- Возврат значения: `return`, `void`, возврат `null` из ссылочных методов.
- Перегрузка: одинаковые имена, разные параметры (количество, тип, порядок), не зависит от возвращаемого типа.
- Ключевое слово `this` в конструкторах для перегрузки.

**Конспект:** 5–10 строк в файл `03_Theory_Core/09_Methods_Overload.md`
