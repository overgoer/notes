## ❌ Ошибки Дня 1

### 1. Синтаксис `for` — НЕВЕРНЫЙ ❌

**Твой код:**
```java
for(i in arr ){  // ❌ ЭТО НЕ JAVA!
    counter++;
}
```

**Правильно в Java:**

```java
// Вариант 1: for-each (для перебора элементов)
for (String element : arr) {
    counter++;
}

// Вариант 2: классический for (по индексу)
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

### 2. Массивы — ошибка в типе ❌

**Твой код:**
```java
Int[] myArray = new Integer[5];  // ❌ Int не существует!
```

**Правильно:**
```java
int[] myArray = new int[5];           // примитивы
Integer[] myArray = new Integer[5];   // объекты (обёртки)
```

**Запомни:** `Int` не существует в Java! Есть `int` (примитив) и `Integer` (класс-обёртка).
