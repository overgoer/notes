## 8 примитивных типов Java

| Тип | Размер | Диапазон | Значение по умолчанию | Пример |
|-----|--------|----------|----------------------|--------|
| `byte` | 1 байт | -128 до 127 | `0` | `byte b = 100;` |
| `short` | 2 байта | -32,768 до 32,767 | `0` | `short s = 1000;` |
| `int` | 4 байта | -2³¹ до 2³¹-1 (~±2.1 млрд) | `0` | `int i = 100000;` |
| `long` | 8 байт | -2⁶³ до 2⁶³-1 | `0L` | `long L = 100000L;` |
| `float` | 4 байта | ~±3.4e-38 до 3.4e+38 | `0.0f` | `float f = 3.14f;` |
| `double` | 8 байт | ~±1.7e-308 до 1.7e+308 | `0.0d` | `double d = 3.14159;` |
| `char` | 2 байта | 0 до 65,535 (Unicode) | `'\u0000'` | `char c = 'A';` |
| `boolean` | ~1 бит | `true` или `false` | `false` | `boolean flag = true;` |

**Примечания:**
- `char` хранит один символ в кодировке UTF-16.
- `boolean` в JVM зависит от реализации, но логически только true/false.
- Литералы: `L` для long, `f` для float, `d` для double (можно опустить).

## Ссылочные типы (Reference Types)

1. **Классы** — `String`, `Integer`, `Double`, любые пользовательские классы.
2. **Массивы** — `int[]`, `String[]`.
3. **Интерфейсы** — `List`, `Map`.

**Особенности:**
- Хранят ссылку на объект в куче (heap).
- Значение по умолчанию — `null`.
- Поддерживают методы и наследование.

## Разница `int` vs `Integer`

| Критерий | `int` (примитив) | `Integer` (обёртка) |
|----------|------------------|---------------------|
| **Тип** | Примитивный | Класс (ссылочный) |
| **Память** | Хранится в стеке (значение) | Хранится в куче (объект) |
| **Значение по умолчанию** | `0` | `null` |
| **Методы** | Нет | `toString()`, `parseInt()`, `compareTo()` |
| **Использование** | Арифметика, счётчики | Коллекции (например, `List<Integer>`), nullable значения |
| **Автоупаковка/распаковка** | Автоматически конвертируется | Автоматически конвертируется |

**Автоупаковка (auto-boxing):**
```java
int primitive = 5;
Integer wrapper = primitive;  // автоупаковка: Integer.valueOf(primitive)
```

**Распаковка (unboxing):**
```java
Integer wrapper = 10;
int primitive = wrapper;  // распаковка: wrapper.intValue()
```

**Важно:** `Integer` может быть `null`, `int` — нет. При распаковке `null` вызовет `NullPointerException`.

## Объявление переменных

```java
// Примитивы
int age = 30;
double price = 19.99;
boolean isActive = true;

// Ссылочные типы
String name = "Ed";
Integer count = 5;
int[] numbers = {1, 2, 3};
```

**Правила именования:**
- camelCase: `userAge`, `totalCount`
- Нельзя начинать с цифры
- Может содержать буквы, цифры, `_`, `$`
- Чувствительны к регистру

## Литералы

```java
int decimal = 100;
int hex = 0xFF;        // 255 в шестнадцатеричной
int binary = 0b1010;   // 10 в двоичной
long big = 1_000_000L; // с подчёркиваниями для читаемости
double sci = 1.23e4;   // 12300.0
char letter = 'A';
char unicode = '\u0041'; // 'A'
String text = "Hello";
boolean yes = true;
```

## Ключевое слово `var` (Java 10+)

```java
var list = new ArrayList<String>(); // тип выводится как ArrayList<String>
var count = 10;                     // int
var message = "Hello";              // String
```

**Ограничения:**
- Только для локальных переменных.
- Должен быть инициализатор.
- Нельзя использовать для параметров методов и полей класса.
