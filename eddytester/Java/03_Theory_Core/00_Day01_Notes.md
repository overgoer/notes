## Ответы на диагностику:
1. **Что такое переменная?** Переменная — это именованная область памяти, хранящая значение определённого типа. Примитивные типы Java: byte, short, int, long, float, double, char, boolean.
2. **Разница между `int` и `Integer`:** `int` — примитив, хранит значение; `Integer` — класс-обёртка, объект, может быть null, имеет методы.
3. **Цикл `for`:** выполняет блок кода повторно. Синтаксис: `for (инициализация; условие; инкремент) { тело }` или `for (тип переменная : коллекция) { тело }`.
4. **Массив:** упорядоченная коллекция элементов одного типа. Создать массив из 5 элементов: `int[] arr = new int[5];` или `int[] arr = {1,2,3,4,5};`.
5. **Что выведет код:** `5`. Значение `x` не меняется при изменении `y`.
