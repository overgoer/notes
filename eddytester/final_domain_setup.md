## Что сделает скрипт автоматизации

1. **Установит и настроит Caddy**:
   - Веб-сервер с автоматическим TLS
   - Reverse proxy на Cloak (порт 8443)
   - Автоматическое получение SSL сертификатов через Cloudflare DNS

2. **Настроит ротацию SNI**:
   - Создаст скрипт `/usr/local/bin/rotate-sni.sh`
   - Настроит cron задание на ротацию каждые 6 часов
   - SNI будет меняться на популярные домены (discord.com, cloudflare.com и др.)

3. **Обновит систему мониторинга**:
   - Добавит проверку доступности домена
   - Создаст тестовую страницу `/var/www/monitor/domain-test.html`

4. **Настроит firewall**:
   - Откроет порты 80 и 443

## Проверка после настройки

### 1. Проверка TLS сертификата
```bash

```
Должен вернуть статус 200 или 404 (главное — без ошибок SSL)

## Устранение неполадок

### Проблема: Домен не разрешается
```bash
dig pupupuuu.com
```
Убедитесь, что возвращается IP `217.144.185.210`

### Проблема: Ошибка SSL
```bash
journalctl -u caddy --no-pager -n 50
```
Проверьте логи Caddy

### Проблема: Caddy не запускается
```bash
systemctl status caddy
caddy validate --config /etc/caddy/Caddyfile
```

### Проблема: Cloak не отвечает через домен
```bash
netstat -tlnp | grep 8443
systemctl status cloak
```
