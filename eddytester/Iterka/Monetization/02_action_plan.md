# План действий: Монетизация Iterka

**Горизонт:** 12 месяцев
**Текущее состояние:** Пре-монетизация (доход = 0₽)

---

## Блок A: Быстрые победы (1-2 недели)

### A1. Создать медиакит
- [ ] Собрать статистику канала в PDF/Google Slides
- [ ] Описать форматы интеграций с ценами
- [ ] Разместить ссылку на медиакит в описании канала
- [ ] Сделать версию для Telegram

**Форматы и цены для медиакита:**
| Формат | Описание | Цена |
|--------|----------|------|
| Преролл | 15-30 сек в начале | 5 000₽ |
| Интеграция нативная | 1-2 мин в середине | 10 000₽ |
| Пост-ролл | 15-30 сек в конце | 3 000₽ |
| Упоминание в описании | Ссылка + текст | 2 000₽ |
| Спонсорство серии (4 выпуска) | Логотип + упоминания | 30 000₽ |

### A2. Запустить Boosty
- [ ] Создать страницу на Boosty
- [ ] Уровни: 300₽ (спасибо), 500₽ (расшифровки), 1000₽ (доступ к чату)
- [ ] Анонсировать в Telegram и YouTube
- [ ] Добавить ссылку в описание каждого видео

### A3. Оживить Telegram
- [ ] Публиковать минимум 3-4 поста в неделю
- [ ] Бэкстейдж со съёмок
- [ ] Анонсы будущих гостей
- [ ] Опросы: кого позвать следующим
- [ ] IT-мемы (юмористический контент)

---

## Блок B: Рост канала (1-3 месяца)

### B1. Увеличить частоту до 2 выпусков/мес
- [ ] Оптимизировать процесс: искать гостей заранее
- [ ] Снимать 2 выпуска за 1 день в студии (экономия на студии)
- [ ] Делегировать монтаж (5 000₽/видео)
- [ ] Нанять ассистента для поиска гостей

### B2. SEO и упаковка
- [ ] Исследовать ключевые слова (карьера IT, собеседование, зарплата и т.д.)
- [ ] Улучшить заголовки (добавить "громкие" имена и цифры)
- [ ] Превью: единый стиль, яркие лица, текст
- [ ] Разбивать длинные видео на отрезки с таймкодами

### B3. Формат контента
- [ ] **ЧЕК-ЛИСТ:** Гость → его путь → ключевые уроки → советы для зрителей
- [ ] Короткие вертикальные версии для Shorts (тизеры интервью)
- [ ] Посты-резюме выпуска на vc.ru / Habr

---

## Блок C: Монетизация (3-6 месяцев)

### C1. Привлечение рекламодателей
- [ ] Зарегистрироваться в **Mave.ads** каталоге
- [ ] Составить список целевых рекламодателей:
  - Образовательные платформы (Яндекс.Практикум, Skillbox, Нетология)
  - IT-компании с HR-брендом (Т-Банк, Авито, Яндекс, Ozon)
  - Инструменты для разработчиков (GitLab, JetBrains, Sentry)
  - Telegram-каналы по IT (купить рекламу у них в обмен на интеграцию)
- [ ] Подготовить шаблон коммерческого предложения
- [ ] Холодные контакты через LinkedIn/email

### C2. Переговоры
- [ ] Первым клиентам — существенная скидка (тестовый режим)
- [ ] Предлагать пакет: 3 выпуска со скидкой 20%
- [ ] Акцент на IT-нишу и лояльную аудиторию
- [ ] Предоставлять UTM-ссылки для отслеживания

### C3. Boosty + консультации
- [ ] Если Boosty даёт 20+ подписчиков — запустить закрытый чат
- [ ] Анонсировать консультации как "разбор карьеры с ведущим подкаста"
- [ ] Цена консультации: 3 000₽/30 мин для начала

---

## Блок D: Масштабирование (6-12 месяцев)

### D1. YouTube AdSense
- [ ] До 4 000 часов просмотра (~20-25 выпусков)
- [ ] После подключения доход: 500 - 6 000₽/мес (вспомогательный)

### D2. Курс "Карьера в IT"
- [ ] Собрать паттерны из интервью в единый гайд
- [ ] Формат: Notion/PDF/видео-уроки
- [ ] Цена: 1 500 - 3 000₽
- [ ] Потенциал: 50-200 продаж = 75 000 - 600 000₽

### D3. Партнёрские программы
- [ ] Реферальные ссылки от гостей (если у них есть курсы)
- [ ] Партнёрка GitPod/Codesandbox и т.д.
- [ ] Книги по IT (партнёрка Литрес)

## Риски и смягчение

| Риск | Вероятность | Смягчение |
|------|-------------|-----------|
| Нет рекламодателей при малых охватах | Высокая | Начать с Boosty и консультаций |
| Канал не растёт | Средняя | Поменять формат, добавить Shorts |
| Выгорание ведущего | Средняя | Делегировать монтаж и поиск гостей |
| Уход гостей к конкурентам | Низкая | Уникальный формат, личное обаяние Эда |
| Блокировка YouTube | Средняя | Развивать Telegram и аудиоверсии |
