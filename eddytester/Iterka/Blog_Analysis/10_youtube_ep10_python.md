## Ключевые темы

| Тема | Описание |
|------|----------|
| **Почему Python №1** | Python — самый популярный язык по версии GitHub (2024). Причины: низкий порог входа, лаконичный синтаксис, огромная экосистема библиотек, скорость разработки |
| **Плюсы Python** | Простота изучения, читаемость кода («как английский текст»), быстрое прототипирование, огромное сообщество, применение в AI/ML, веб-разработке, автоматизации |
| **Минусы Python** | Медленнее компилируемых языков (C++, Rust) в runtime, не подходит для gamedev (кроме серверной части), не используется в нативной мобильной разработке |
| **Сравнение с другими языками** | Python vs Java (быстрее разработка, проще), Python vs C++ (медленнее, но быстрее писать), Rust — популярен, но малая экосистема мешает массовости |
| **Будущее Python** | Python на пике популярности, замены пока не видно. Огромная инфраструктура и экосистема — барьер для любого конкурента |
| **Путь новичка** | «От простого к сложному»: аналогия с вождением (автомат → механика). Сначала Python и веб-разработка, потом переход на более сложные языки |
| **Выбор первого проекта** | Брать то, что нужно в жизни. Пример: калькулятор стоимости владения авто вместо абстрактного to-do листа |
| **Методология обучения** | Пошаговое наращивание: маленькие задания → код переходит из одного в другое → проект обрастает фичами. Не туториал, а самостоятельное решение задач |
| **Не бояться переписывать** | Код можно и нужно переписывать 3-4 раза, каждая итерация делает его оптимальнее |
| **Грейды и опыт** | Грейд субъективен и зависит от компании. Опыт = не годы в трудовой, а реальные навыки. Можно за 3 года стать круче «синьора» с 10-летним стажем |
| **Собеседования** | Практика — лучший способ подготовки. Ходить на собесы, записывать их, анализировать. Разница РФ vs Европа/Израиль |
| **Личная трансформация Егора** | Семейный кризис → поездка на «Метаморфозы» к Петру Осипову → переосмысление: сбрил усы, начал бегать, вести блог, развёлся, переехал |
| **Образовательная платформа** | Собственная платформа курса (не Stepik/Coursera) — показатель экспертности. Интеграция с GitHub, автоматическая проверка ДЗ |
| **Инфоцыгане** | Егор не получал обвинений в инфоцыганстве. Важно: бесплатная вводная часть, ограничение по времени (3 месяца), 70% доходимость |

## Ключевые цитаты

> «Python — это самый популярный язык программирования по версии GitHub 2024 года»

> «На C++ тебе надо месяц что-то писать, чтобы получилось. На Python то же самое ты напишешь за пару дней»

> «Скорость выхода на рынок — она решает»

> «Python как будто текст читаешь. Код на Python сочетается с тем, как мы говорим»

> **Про выбор первого проекта:** «Возьми то, что тебе нужно в жизни. Мне нужен был калькулятор стоимости машины — я его написал»

> **Про переписывание кода:** «Не бояться переписывать. Я одну задачу могу сделать 3-4 раза, просто написав заново. И третья-четвёртая итерация уже оптимальна»

> **Про грейды:** «Грейд — это субъективная вещь внутри компании. Опыт — это не сколько у тебя в трудовой книжке, а что ты своими руками можешь делать»

> **Про собеседования в Amazon:** «4 часа непрерывного лайфкодинга с 8 людьми. Если компания так проявляет себя на собеседовании — работа у них весёленькая»

> **Про дисциплину:** «Если что-то хочешь освоить — делай это каждый день. Когда нет выбора — следовать легче, чем когда есть выбор»

> **Про трансформацию:** «Появилось больше смелости. Смелость переехать, развестись, начать новое. И энергия на это»

> **Про ценность курса:** «Платят за структурирование информации и донесение простым языком. В интернете есть всё, но в этом и проблема — тебе всё не надо»

> **Аналогия с вождением:** «Сначала учись на автомате (Python), потом пересаживайся на механику (C++/Rust/Go). Когда на автомате освоил руль и тормоз — добавить переключение передач не составляет труда»
