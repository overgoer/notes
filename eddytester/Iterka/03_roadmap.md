## 🎯 Стратегия

**Фокус:** не пытаться быть "ещё одним IT-подкастом", а стать **голосом инженеров, у которых нет своего канала**.

### Тезис
Крутых инженеров-блогеров с аудиторией — единицы. Но есть **сотни Senior-инженеров и тимлидов из бигтеха**, у которых нет YouTube-канала, но есть имя в индустрии, экспертиза и история. Iterka может быть **их первой трибуной**.

### Необычный карьерный путь (концепция)
Ценность не в имени, а в **заголовке**. Собирают просмотры за счёт контраста.
| Концепция | Потенциальный герой |
|-----------|---------------------|
| Повар → Senior Developer | Найти через истории вката |
| Военный → Team Lead | Есть кейсы VK / Яндекс |
| Филолог → CTO | + |
| 50 лет → первый код | Возрастная история |
| Из деревни в FAANG | Из региона в международную компанию |

### "Провалы сеньоров" — компиляция коротких интервью
Формат: 5-10 минут с разными Senior'ами про их **крупнейшие IT-катастрофы**. 
- "День, когда я удалил продакшен"
- "Я залил тестовые данные в боевую базу"
- "Мой первый код ушёл в прод без ревью"

**Почему залетит:**
- Релатабельно для каждого айтишника
- Юмористически-просветительский формат
- Никто из подкастов этим не занимается
- Клиповые нарезки = виральные шортсы

**Концепция:** IT-викторина типа "Своя игра", но динамичнее, с юмором.
- Участники отвечают на IT-вопросы
- Зрители играют вместе (в уме или в чате)
- Кнопки, табло, оформление
- Юмористически-просветительский формат

**Бюджет:** пока нет. Отложено на конец года.

## ⚡ Форматы роста (дополнительно)

1. **Shorts-нарезки** — с каждого выпуска 3-5 шортсов (виральные цитаты)
2. **Реакции на IT-новости** — 15 минут раз в 2 недели (дёшево, регулярно)
3. **"Провалы сеньоров"** — компиляция 5-10 коротких интервью
4. **QA-спецвыпуски** — связка с Eddytester, воронка подписчиков
5. **Необычный карьерный путь** — герои с контрастным бэкграундом
